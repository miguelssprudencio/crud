# ✅ Focus — Lista de Tarefas

Um aplicativo moderno e responsivo para gerenciamento de tarefas, desenvolvido para ajudar na organização das atividades do dia a dia de forma simples, intuitiva e eficiente.

O projeto foi desenvolvido utilizando apenas **HTML5**, **CSS3** e **JavaScript (Vanilla)**, priorizando desempenho, organização de código, responsividade e uma excelente experiência de uso em qualquer dispositivo.

---

# 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* LocalStorage
* SVG Icons

---

# ✨ Funcionalidades

* ✅ Cadastro de tarefas
* ✏️ Edição de tarefas
* 🗑️ Exclusão de tarefas
* ✔️ Marcar tarefas como concluídas
* 📂 Organização por categorias
* 🚩 Definição de prioridades
* 📅 Adição de data de vencimento
* 🔄 Persistência dos dados com LocalStorage
* 🌙 Alternância entre tema claro e escuro
* 📱 Layout totalmente responsivo
* 🎨 Interface moderna e intuitiva
* ⚡ Alto desempenho e carregamento otimizado
